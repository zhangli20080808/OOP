1. 面向过程编程

    什么是面向过程编程
    什么是面向过程编程：
    面向过程就是使用函数将一个大的功能分成很多小功能，每个功能称之为一个过程，这就是面向过程编程。
    我们可以考虑使用函数将原来写在一起的一坨的代码拆分成一个一个的独立的小功能
